# Volleyball — Preview

Public **demo / preview** build for [PixelSports Volleyball](https://github.com/subtiliorars-sys/Volleyball) — joyous pixel-art beach volleyball and the founding game on the PixelSports slate.

## What this repo is

This repository holds a **sanitized, static web snapshot** (GitHub Pages) of a playable slice — not the full private product codebase. Open `index.html` or the published Pages URL to try beach volleyball: solo warm-up or same-screen two-player.

**Not included here:** source TypeScript, design docs, build pipeline, or unreleased features from the HQ repo.

## Main product (private)

Full development lives in the private HQ repo:

- [subtiliorars-sys/Volleyball](https://github.com/subtiliorars-sys/Volleyball) *(private — org members only)*
